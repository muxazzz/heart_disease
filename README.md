# Прогнозирование сердечного приступа
Приложение для прогнозирования риска сердечного приступа с помощью модели машинного обучения на основе медицинских, поведенческих и биохимических данных пациентов.


# FastAPI + Docker

FastAPI-приложение в Docker-контейнере.

## Локальный запуск

```bash
docker build -t fastapi-demo .
docker run -d -p 8000:80 fastapi-demo
```

Приложение доступно по адресу
http://localhost:8000
